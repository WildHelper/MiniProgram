## 安装

先运行 `npm install`；然后进入微信开发者工具，选择 `工具 - 构建 npm`

下载 `echart.js` 到 `./miniprogram/ec-canvas` 目录中

该项目有意将 `.idea` 提交。使用 WebStorm/PhpStorm 编辑文件时会原地将 TypeScript 文件编译为 JavaScript。
